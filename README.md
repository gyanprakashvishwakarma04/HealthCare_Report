# Healthcare Analytics Dashboard (Power BI)

## 📌 Project Overview
This Power BI project provides a comprehensive, data-driven analysis of healthcare operations, financial performance, and patient demographics. The interactive dashboard enables hospital administrators and healthcare professionals to optimize resource allocation, track revenue streams, and monitor patient health trends effectively.

🎥 **Project Demo:** Refer to the file `Screen Recording 2026-06-24 044621.mp4` in this repository to see the full interactive walkthrough.

---

## 📊 Key Insights & Metrics (KPIs)
* **Patient & Resource Tracking:** Analyzed **40.24K total patients** with a balanced **1.00 Average Patient per Doctor** ratio, ensuring stable hospital resource distribution.
* **Financial Performance:** Monitored **₹1.42 Billion in Total Revenue**, with an **Average Cost per Patient of ₹25.54K**.
* **Demographics:** Mapped operational volume across patient age groups (Average age: ~51.5 years) and gender distributions.
* **Clinical Profiles:** Segmented total patients across critical medical conditions (Arthritis, Diabetes, Obesity, Hypertension, Asthma, Cancer) and tracked admission types (Emergency, Urgent, Elective).

---

## 🛠️ Tech Stack & Skills Demonstrated
* **Power BI Desktop:** Dashboard design, canvas layout, and interactive data visualization.
* **Power Query (ETL):** Data cleaning, profiling, type conversion, and structural transformations.
* **DAX (Data Analysis Expressions):** Created custom measures for tracking hospital KPIs, running totals, and dynamic averages.
* **Data Modeling:** Established relationships between patient demographics, financial variables, and clinical timelines.

---

## 🚀 Key Features of the Dashboard
1. **Dynamic Slicers:** Filter data dynamically by *Date of Admission* (2019–2024) or specific *Months*.
2. **Interactive Toggles:** Instantly switch views between **Male** and **Female** patient cohorts using custom visual buttons.
3. **Insurance Analysis:** Visualized patient distribution across top health insurance providers (Medicare, Cigna, Aetna, Blue Cross, UnitedHealthcare).
4. **Test Results Breakdown:** Pie chart tracking the proportion of **Normal**, **Abnormal**, and **Inconclusive** medical test results.
